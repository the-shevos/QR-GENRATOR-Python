# QR-GENRATOR-Python

A simple Python application to generate QR codes from any text or URL. Users can customize the QR code colors and save it as an image.

---

## Description

This project allows you to quickly create QR codes for URLs or any text input. You can:

- Enter any text or URL
- Choose the filename for the QR code image
- Customize the QR code color and background color
- Save the QR code as a PNG image

It's built using Python and the `qrcode` library with PIL support.
